# CURSO_ASIMOV

Projeto de estudos com **LangChain**, cobrindo ChatModels, mensagens (sistema/usuário/IA), Few-Shot e integração com Ollama e Hugging Face.

## Requisitos

- **Python** 3.11+
- **Ollama** instalado e rodando (para os exemplos com modelo local)
- (Opcional) Conta no **Hugging Face** e token para usar modelos via API

## Instalação

### 1. Clone ou acesse o repositório

cd CURSO_ASIMOV
